Riot API

GitHub

Riot provides easy access to their API for developers to collect data based on their games. As a way to learn Javascript and databases (nedb) I created this to gather data about TFT to find out more specific data about the game.

TFT, Team Fight Tactics, is an auto battle which players buy units and have them fight and each player controls a little legend. In decided to gather data on which little legend tends to win the most game. I used node with node-fetch in order to fetch the data from the api with a interval set to not go over my 20 calls per 2 minutes Riot allows. I stored the data in nedb, a light weight local mongodb varient, and then processes the data and turns it into xlsx to turn into graphs and more readable charts.
